CheckSolat - Malaysia Prayer Times
Live site: https://zdarkchoco.github.io/CheckSolat/
CheckSolat is a client-side web application that displays accurate daily Islamic prayer times for all zones in Malaysia. The application fetches data from a public API that mirrors the official schedule provided by Jabatan Kemajuan Islam Malaysia (JAKIM).
Users can view the schedule by manually selecting their specific state and district zone from a dropdown menu. Alternatively, the application includes a GPS feature that utilizes the browser's Geolocation API to detect the user's coordinates and automatically determine the correct JAKIM zone.
The project is built entirely as a single-page application using plain HTML, Vanilla JavaScript, and Tailwind CSS via a Content Delivery Network (CDN). Because it does not rely on a JavaScript framework like React or a build pipeline, it deploys directly and natively to GitHub Pages without requiring compilation steps or a package manager.
